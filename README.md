# 使用說明📖

**請事先安裝好[python3](https://www.python.org/)**

* **打開run.bat**
* **輸入"y"安裝pip packages**
* **輸入"y"輸入您的bot token**
* **成功開始執行!**

_**若是已經安裝及設定後再次開啟可直接輸入"n"略過上述步驟**_
